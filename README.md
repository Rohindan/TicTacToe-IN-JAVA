# Tic Tac Toe in Java 🎮

A simple **Tic Tac Toe** game built using **Java**. This project was created to practice Java programming concepts and improve problem-solving skills.

## ✨ Features

* Two-player gameplay (Player X vs Player O)
* Simple and clean interface
* Detects the winner
* Detects draw/tie situations
* Restart and play again

## 🛠️ Technologies Used

* Java
* IntelliJ IDEA

## 📁 Project Structure

```
TicTacToe/
├── src/
│   └── (Java source files)
├── .gitignore
└── README.md
```

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/Rohindan/TicTacToe-IN-JAVA.git
```

2. Open the project in IntelliJ IDEA.
3. Run the main Java file.
4. Enjoy playing Tic Tac Toe!
